Tutorial: molecular genetic analysis of Dengue virus

This tutorial described how to extend the Flaviviridae resource for individual viruses, using dengue virus (DEN) as an example.

Please note that we could equally build a stand-alone Dengue project, and instructions for doing this are also provided in this project.

Building the Dengue project within Flaviviridae-GLUE means that our Dengue project will have 'knowledge' of the entire Flaviviridae family, and can therefore support comparisons between Dengue and other flaviviruses. Depending on the sequence analysis context, this may or may not be a useful feature. This tutorial focuses on. It comprises the following steps:

  1. Downloading selected sequences from GenBank.
  2. Extracting isolate data from GenBank files.
  3. Filtering sequence data to obtain a valid analysis set.
  4. Defining virus reference sequences and associating them with annotations.
  5. Constructing an alignment using a codon-aware method.
  6. Mapping feature coverage of sequence members within an alignment.
  7. Reconstructing annotated, midpoint-rooted phylogenetic trees.
  8. Using GLUE projects as a basis for collaborative research and resource development.

Step 1: creating a basic sequence database for Dengue virus

The first step is to build a database of Dengue sequences from those published in GenBank. The GLUE framework can be used to quickly build a sequence database of published sequences for any virus. Let's do this now for Dengue.
