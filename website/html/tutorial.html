Exporting protein and polypeptide sequences

In GLUE, protein-coding features are explicitly defined on reference sequences. Protein-coding features of reference sequences can be exported using a 'fastaProteinExporter' module as follows.

First create the module. We will use the -t option to create the module with a default specification. 

create module -t fastaProteinExporter
OK
(1 Module created)
Mode path: /project/flaviviridae

The module can now be used to export NS5 protein sequences for all flavivirus genus reference sequences in which the NS5 protein has been annotated. 

Mode path: /project/flaviviridae
GLUE> module fastaProteinExporter
OK

Mode path: /project/flaviviridae/module/fastaProteinExporter
GLUE> export reference -f NS5 -w "sequence.genus = 'Flavivirus'" -p
